#  Vite +  Vue 3 Blogs  

A simple blog web built with Vite and Vue 3 Composition API using Tailwind CSS. 

## Features
- Vue 3 Composition API
- Post Create 
- Post Read
- Post Update
- Post Delete

## Installation in local
```bash
git clone https://github.com/AbdurRaahimm/vue-blogs.git
```
```bash
cd vue-blogs
```
```bash
npm install
```
```bash
npm run dev
```
## Json Server run
```bash
npm run server
```
